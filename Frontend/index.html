Välkommen till Recept roulette!:

Tryck på knappen "Recept roulette!" för att få fram ett random recept som du sen kan spara!

Du kan även välja en kategori livsmedel nedanför för att få upp till 20 recept som inspiration!

Dessa kan du sedan lägga till egna notiser till och se recept samt indigrenser, skulle det smaka "skit" så kan du ta bort receptet!
